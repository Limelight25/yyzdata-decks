Install it

Gmail (web): click Copy signature above, then in Gmail: See all settings → General → Signature → Create new → paste → set it under "For new emails use" → Save Changes at the bottom of the page.

Gmail app on your phone

Nothing to do. Once it is set on the web, the Gmail app uses it automatically. This is the only phone client that keeps the logo.

Apple Mail (Mac)

Mail → Settings → Signatures → select your account → + → first uncheck "Always match my default message font", then click into the right pane and paste.

iPhone Mail app

Settings → Mail → Signature. Apple strips images and formatting there, so use the plain version button above. The logo will not carry in iPhone Mail; use the Gmail app for the full look.
